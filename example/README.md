# Example

This directory contains example applications demonstrating Flart's capabilities.

## Running the Examples

```bash
cd example
dart run build_runner serve
```

Then open your browser to `http://localhost:8080`

## Examples

### 1. Hello World
Basic Flart application showing the minimal setup.

### 2. Widget Showcase
Comprehensive showcase of all available widgets.

### 3. Form Example
Complete form with validation and submission.

### 4. Animation Demo
Various animation examples.

### 5. Media Gallery
Video, audio, and YouTube player examples.

## Creating Your Own Example

1. Create a new `.dart` file in this directory
2. Import Flart: `import 'package:flart/flart.dart';`
3. Create your app and run it with `runApp()`
4. Build and serve with `dart run build_runner serve`
